# 김동규 Backend Portfolio

Spring Boot 기반 백엔드 개발과 배포·운영 경험을 정리한 포트폴리오입니다.

## Main Experience

- Lion Connect 인재풀 매칭 서비스 백엔드 개발
- Spring Boot / Java / MySQL / JPA / QueryDSL 기반 REST API 구현
- JWT 인증, S3 Presigned Upload 파일 업로드 구현
- AWS EC2/RDS/S3, Docker, GitHub Actions, CloudWatch 기반 배포·운영 경험
- Auto Scaling Group 기반 가변 서버 운영 및 무중단 배포 고려 경험
